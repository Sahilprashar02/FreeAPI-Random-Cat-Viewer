## ✨ Features

- **Dynamic Fetching**: Instantly load a new cat image with a single click.
- **Premium Design**: Dark-themed UI with glassmorphism effects and smooth transitions.
- **Responsive Layout**: Optimized for mobile, tablet, and desktop screens.
- **Loading States**: Shimmer effects and button loaders for a better UX.
- **SEO Ready**: Proper semantic HTML and meta descriptions.

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Icons**: Font Awesome 6
- **Typography**: Google Fonts (Outfit)
- **API**: [FreeAPI](https://freeapi.app/)

## 🚀 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/catsnap-viewer.git
   ```
2. Navigate to the project directory:
   ```bash
   cd catsnap-viewer
   ```
3. Open `index.html` in your favorite browser.

## 📡 API Used

This project uses the **FreeAPI** Random Cat endpoint:
`https://api.freeapi.app/api/v1/public/cats/cat/random`
